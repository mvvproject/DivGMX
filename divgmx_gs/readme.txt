Конфигурация звуковой карты General Sound для spectrum-совместимых компьютеров с шиной ZX-BUS.

- Полная программная совместимость с General Sound;
- Гибкая архитектура, основанная на fpga (Cyclone IV EP4CE10);
- Собственная память 4МБ из 32МБ ОЗУ, 32КБ из 8МБ flash-ROM; 
- Расширенные способы адресации памяти: 2 окна проецирования, полное исключение ROM из адресного пространства;
- Z80 20МГц;
- Интерфейс SD-card с аппаратным SPI для пересылки данных; 
- 8 аппаратных звуковых каналов с громкостями;
