NZX (сборка 20170429)
Видео https://youtu.be/R0lOSh_YAmU

Тестовая спектрум-подобная двухпроцессорная система на процессорах nextZ80 по 33.3МГц каждый, совместно делят SDRAM.
Процессоры пока работают каждый в своем адресном пространстве по 64К и со своим видео буфером 6912 байт.
Для возможности тестирования поддерживается USB клавиатура с возможность переключения для каждого процессора (F1-F2).
Изображение двух спектрумовских экранов наложено на общее разрешение 640х480@60Hz, планирую использовать для тестирования видео режима 640х480 256 цветов.

Специальные клавиши на клавиатуре:
0 - 5	Картинка в формате bmp 640x480 256 цветов
F1	Клавиатура для CPU0
F2	Клавиатура для CPU1
F10	Просмотр GFX экрана
PrtScr	Сброс

Порта:
xxFE	D2-D0 = цвет бордюра
08	GR_BASE0 адрес(15..0) начала видео буфера
09	GR_BASE1 адрес(31..16) начала видео буфера
0A	GR_PAGE видео страница доступна по адресу C000-FFFF
0B	GR_PALETTE адрес даступа
