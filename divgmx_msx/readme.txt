OneChipMSX (сборка 20170330)
основана на http://www.caro.su/msx/ocm_de1.htm

1) Отформатируйте SD карту в FAT16 на PC
2) Запишите на неё первым файлом bios_m2p.rom из архива sd.zip
3) Запишите остальные файлы из архива sd.zip
4) Подключите к плате DivGMX кабель HDMI, USB клавиатуру, аудио и включите питание платы
5) Откройте в Quartus Programmer файл divgmx_msx20170330.jic
6) Запрограммируйте DivGMX и отключите питание DivGMX
7) Вставте SD карту в DivGMX и включите питание
8) Для запуска SYMBOS наберите symb и нажмите ENTER, режим 512х212(16 цветов) пока не работает, выбирете в окне Display>Modi 512x212(4 colour)
9) Для запуска игры наберите cd games и нажмите ENTER, набирите dir, набирите romload mg2-uk /s

Управление указателем в SYMBOS с помощью клавиш F6 + курсорные клавиши (вверх, вниз, влево, вправо, пробел)

Специальные клавиши на клавиатуре:
Page Up		Добавить громкость звука
Page Down	Уменьшить громкость звука
F9		Добавить уровень звука в канале PSG
F9+SHIFT	Уменьшить уровень звука в канале PSG
F10		Добавить уровень звука в канале SCC
F10+SHIFT	Уменьшить уровень звука в канале SCC
F11		Добавить уровень звука в канале OPLL
F11+SHIFT	Уменьшить уровень звука в канале OPLL
F12		Переключение тактовой частоты процессора
